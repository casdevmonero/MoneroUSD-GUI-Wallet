  SPEND-AUTH & LINKABILITY (SAL) PROOF

  The SAL proof demonstrates:

    3. Linkability: the key image is deterministic per output

  Components per input:
    ┌──────────────────────────────────────────────────────────┐

    │  C_tilde = rerandomized Pedersen commitment              │
    └──────────────────────────────────────────────────────────┘

  Balance Proof:
    Σ pseudo_outs == Σ output_commitments + fee_commitment
    This is verified separately via Generalized Bulletproofs+.

================================================================================
  GENERALIZED BULLETPROOFS+ (Range Proofs)
================================================================================

  Each output commitment C = aG + bH proves b ∈ [0, 2^64)
  without revealing the amount. FCMP++ integrates range proofs
  into the same arithmetic circuit as the membership proof,
  reducing total proof size vs. separate Bulletproofs+.

  Proof aggregation: 2 inputs batched into single circuit proof
